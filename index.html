📄 Aula 1: O que é HTML?

Bem-vindo à web!

HTML (HyperText Markup Language) não é uma linguagem de programação, mas sim uma linguagem de marcação. Ela diz ao navegador como estruturar o conteúdo da página.

Pense no HTML como o "esqueleto" de um site. O CSS será a pele/roupa, e o JavaScript (ou Python no back-end) será o cérebro.

📄 Aula 2: Estrutura Básica de uma Página

O Esqueleto HTML

Todo arquivo HTML começa com uma estrutura padrão. Veja abaixo:

<!DOCTYPE html>
<html lang="pt-BR">
<head>
    <meta charset="UTF-8">
    <title>Minha Primeira Página</title>
</head>
<body>
    <h1>Olá, Mundo!</h1>
</body>
</html>
  • <!DOCTYPE html>: Avisa que é HTML5.
  • <head>: Configurações invisíveis (título, idioma).
  • <body>: O que o usuário realmente vê na tela.

📁 Módulo 2: Estruturas de Controle
📄 Aula 3: Condições (IF / ELSE)

Usamos o if para tomar decisões no código.

nota = 7.5

if nota >= 7.0:
    print("Aprovado!")
elif nota >= 5.0:
    print("Recuperação.")
else:
    print("Reprovado.")
📄 Aula 4: Laços de Repetição (FOR)

O for serve para repetir uma ação várias vezes.

# Vai contar de 0 até 4
for numero in range(5):
    print("Contagem:", numero)

📁 Módulo 1: Criando um Servidor Web
📄 Aula 1: Introdução ao Flask

Agora vamos usar Python para criar um site de verdade usando o micro-framework Flask. Primeiro, instale no terminal: pip install flask

from flask import Flask

app = Flask(__name__)

@app.route("/")
def home():
    return "<h1>Meu site feito em Python!</h1>"

app.run()
📄 Aula 2: Entregando seu arquivo HTML

Em vez de escrever HTML dentro do Python, usamos o render_template para carregar aquele arquivo HTML que criamos no Curso 1!

from flask import Flask, render_template

app = Flask(__name__)

@app.route("/")
def index():
    # Isso vai ler o arquivo index.html da pasta 'templates'
    return render_template("index.html")

app.run()
