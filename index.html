🚀 Curso Completo de Programação: Python & HTML

Bem-vindo ao repositório oficial do curso! Aqui você vai aprender a estruturar páginas web com HTML5 e criar lógicas de programação robustas e backends poderosos com Python.

📋 Sobre o Curso

Este curso foi desenhado para quem quer sair do zero e se tornar um desenvolvedor capaz de construir interfaces limpas e sistemas funcionais. Nós combinamos a simplicidade do HTML com a versatilidade do Python.

🛠️ O que você vai aprender

Módulo 1: Fundamentos do HTML5

  • Estrutura básica de uma página (<html>, <head>, <body>).
  • Semântica web para SEO e Acessibilidade.
  • Criação de formulários interativos e validações.
  • Links, imagens, tabelas e iframes.

Módulo 2: Dominando o Python

  • Sintaxe básica, variáveis e tipos de dados.
  • Estruturas de controle (if/else, for, while).
  • Funções, Listas, Dicionários e Tratamento de Erros.
  • Programação Orientada a Objetos (POO).

Módulo 3: O Projeto Final

Vamos unir os dois mundos usando um micro-framework web (como Flask ou FastAPI). Veja um exemplo do que construiremos:

from flask import Flask, render_template

app = Flask(__name__)

@app.route('/')
def home():
    # Renderizando nosso HTML com o poder do Python!
    return render_template('index.html', titulo="Meu Primeiro App")

if __name__ == '__main__':
    app.run(debug=True)

💻 Pré-requisitos

  • Nenhum conhecimento prévio de programação é exigido!
